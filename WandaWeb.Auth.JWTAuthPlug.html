      View Source WandaWeb.Auth.JWTAuthPlug (Wanda v1.2.0)

      Plug responsible for reading the JWT from the authorization header and validating it.

      If the token is valid, the user_id is added to the private section of the connection. If the token is invalid, the connection is halted with a 401 response.
